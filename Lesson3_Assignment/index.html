Git Add

The git add command can be is part of the first few steps that allow you start the git cycle. Git Add, adds all the files that were changed since the last back up to the staging area.

Git Commit

After the file is in the staging area it's time to commit. Git Commit the changes to the repository. Typically if you include -m followed by a single quotations, you allow an opportunity to add a description of the change.

Git Push

Then we Git Push! Push the changes in the local repository into the remote repository. This allows you to always have multiple copies of updated versions regardless if you are at your local computer or not.
